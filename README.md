# PredictX - Web3 Sports Prediction Platform

A modern, gamified sports prediction platform built on Web3 technology where users predict outcomes of real sports events like Formula 1 races and football matches.

## Features

### 🎯 Core Features
- **Skill-Based Predictions**: No gambling, just pure sports knowledge and strategy
- **Multi-Sport Support**: Formula 1 races and Football matches
- **Real-Time Events**: Live and upcoming events with prediction locking
- **Blockchain Integration**: Built on BNB Chain for transparent rewards

### 🏆 Gamification
- **Points & Ranking System**: Earn points for accurate predictions
- **Global Leaderboards**: Daily, Weekly, Monthly, and All-Time rankings
- **NFT Achievement Badges**: Unlock unique NFTs for milestones
- **Platform Coins**: Earn cryptocurrency rewards
- **Seasonal Competitions**: Compete for exclusive end-of-season rewards

### 📱 User Experience
- **Responsive Design**: Optimized for desktop and mobile devices
- **Dark Mode Theme**: Eye-friendly dark interface with neon accents
- **Interactive Dashboard**: Real-time stats, charts, and predictions
- **Personal Profile**: Track history, accuracy, and achievements
- **Wallet Integration**: Connect Web3 wallet for blockchain features

## Tech Stack

- **React 18** with TypeScript
- **React Router 7** for navigation
- **Tailwind CSS v4** for styling
- **Recharts** for data visualization
- **Lucide React** for icons
- **Motion** for animations
- **BNB Chain** for blockchain integration

## Pages

1. **Landing Page** - Hero section with platform overview and CTAs
2. **Dashboard** - User stats, upcoming events, active predictions
3. **Predictions** - Browse and predict on F1 and Football events
4. **Leaderboard** - Global rankings with podium display
5. **Rewards** - Coin balance, NFT collection, reward history
6. **Profile** - Personal stats, prediction history, earned badges

## Getting Started

The application is built with Vite and can be run with:

```bash
npm run build
```

## Key Interactions

- **Connect Wallet**: Connect your Web3 wallet (BNB Chain)
- **Make Predictions**: Select events and submit predictions before they lock
- **Earn Rewards**: Gain points and coins for accurate predictions
- **Collect NFTs**: Unlock achievement badges by completing milestones
- **Climb Leaderboard**: Compete globally for top rankings
