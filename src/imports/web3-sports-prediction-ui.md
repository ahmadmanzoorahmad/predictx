Design a modern Web3 gamified sports prediction platform UI for desktop and mobile.

Product concept:
A skill-based prediction platform (not gambling) where users predict outcomes of real sports events like Formula 1 races and football matches. Users earn points, reputation, and platform coins for accurate predictions. The platform runs on BNB Chain and uses blockchain for reward distribution, NFTs, and seasonal leaderboards.

Style:
Clean, modern Web3 dashboard with esports + sports aesthetics. Dark mode primary theme with neon accents (yellow / electric blue / purple). Premium tech startup feel similar to fantasy sports apps mixed with Web3 dashboards.

Main Screens to Design:

1. Landing Page
Hero section with headline:
"Predict. Compete. Earn."
Subtext explaining skill-based prediction platform.
CTA buttons: Connect Wallet, Start Predicting
Visuals of sports predictions, leaderboard graphics, and blockchain elements.

2. User Dashboard
User stats panel:
- Prediction accuracy %
- Total predictions
- Points earned
- Ranking position
Widgets showing:
- Upcoming events
- Active predictions
- Earned coins
Leaderboard preview.

3. Prediction Interface
Event cards for sports events such as:
- Formula 1 race
- Football match
Each card shows teams/drivers, event time, prediction options.

Prediction UI examples:
- Race winner
- Podium finish
- Match score
- First goal

Lock predictions before event start.

4. Leaderboard Screen
Global leaderboard with:
- Player ranking
- Points
- Accuracy rate
- Badges or NFTs earned.

5. Rewards Page
Display rewards earned:
- Platform coins
- NFT badges
- Seasonal trophies.

Include visual NFT card designs for achievements like:
"F1 Master"
"Football Oracle"

6. Profile Page
User profile showing:
- Prediction history
- Accuracy chart
- Earned NFTs
- Season ranking.

7. Mobile Version
Responsive mobile layout for key screens:
Dashboard, Prediction page, Leaderboard.

UI Components:
- Sports event cards
- Prediction buttons
- Leaderboard tables
- NFT reward cards
- Progress bars for accuracy
- Wallet connect button

Design feel:
Futuristic sports analytics + Web3 gaming platform.

Include placeholder logos, driver icons, football team icons, and charts.

Use modern typography, glowing UI elements, and clean card-based layout.
