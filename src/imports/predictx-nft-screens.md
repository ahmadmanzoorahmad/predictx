Design additional UI screens for a Web3 gamified sports prediction platform called **PredictX**.
PredictX is a skill-based prediction platform where users forecast outcomes of real-world sports events like Formula 1 races and football matches. Users earn points and **NFT achievement cards** for accurate predictions. These NFTs are stored in the user’s wallet and can be traded in an internal marketplace.

Design Style:
Modern Web3 sports gaming interface. Dark theme with neon accents (yellow, electric blue, purple). Premium esports + blockchain dashboard aesthetic.

Add the following new screens and components:

1. **NFT Achievement Card Design**
   Create collectible NFT card layouts representing prediction achievements.

Card elements:

* Card title (example: "Perfect Podium Prediction")
* Event name (example: "Monaco Grand Prix")
* Date of event
* Player username
* Wallet address (short form)
* Rarity level badge (Common, Rare, Epic, Legendary)
* Card artwork with sports theme (Formula 1 cars, football stadium lights)
* PredictX logo
* Unique card ID
* Animated glowing rarity border

Card style inspiration:
Sports trading cards + futuristic Web3 NFT collectible cards.

2. **NFT Rewards Screen**
   A page where users see their earned NFTs.

Sections:

* Grid layout of collectible cards
* Filters by rarity (Common / Rare / Epic / Legendary)
* Sort by newest / value / rarity
* Button: "View on Blockchain"
* Button: "List on Marketplace"

3. **NFT Marketplace**
   Design a marketplace where users buy and sell PredictX NFT cards.

Marketplace UI components:

* Featured rare cards
* Trending cards
* Recently sold cards
* Price displayed in crypto (BNB)
* Card preview modal
* Buy button
* Place offer button
* Seller wallet preview
* Card rarity indicator

Include marketplace filters:

* Sport type (F1 / Football)
* Card rarity
* Price range

4. **NFT Detail Page**
   When a card is opened:

Show:

* Large card artwork
* Owner wallet
* Event details
* Achievement description
* Rarity level
* Transaction history
* Current market price
* Buy / Sell / Transfer buttons

5. **User Collection Page**
   Profile page showing user NFT collection.

Sections:

* Total cards owned
* Rarity distribution chart
* Best achievements
* Featured card display

6. **Wallet Interaction**
   Add Web3 wallet interaction elements:

* Connect Wallet button
* NFT mint confirmation popup
* Transaction success animation
* "NFT added to wallet" notification

Design Requirements:

* Use card-based layouts
* Include sports imagery (Formula 1 cars, football stadium lights)
* Add glowing neon borders for rare NFTs
* Include placeholder icons for wallet addresses
* Ensure mobile responsive layout for NFT screens
* Maintain consistent PredictX branding and dark futuristic UI.
