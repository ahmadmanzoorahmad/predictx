Extend the existing design of the Web3 sports prediction platform PredictX by adding a complete NFT achievement and marketplace system.
PredictX is a skill-based sports prediction platform where users predict outcomes of real-world sports events and earn NFT achievement cards for correct predictions.
The NFT cards represent prediction achievements and are stored in the user’s wallet. These NFTs can be traded inside a marketplace.
Use the same visual style already used in the design:
dark Web3 theme with neon accents and futuristic card layouts.
Add the following new screens and components:
NFT Achievement Card Design
Create collectible NFT card designs representing prediction achievements.
Each card must include:
PredictX logo
Card title (example: Perfect Podium Prediction)
Event name (example: Monaco Grand Prix)
Achievement description
Player username
Wallet address short form
Date of achievement
Rarity level badge
Rarity levels:
Common
Rare
Epic
Legendary
Design cards inspired by sports trading cards combined with futuristic Web3 NFT collectibles.
Use glowing borders and holographic textures depending on rarity.

"NFT successfully minted to your wallet"
